in attesa di dati…
Come caricare i dati. Il browser non può scaricare da plugins.qgis.org (CORS). Scarica uno o più feed con curl, poi trascinali insieme qui (o usa Carica file XML, selezione multipla). Caricando più feed di versioni diverse vengono fusi: per ogni plugin si tiene la versione più alta. Esempio per avere sia i plugin QGIS 3 sia i nativi QGIS 4:
curl -s "https://plugins.qgis.org/plugins/plugins.xml?qgis=3.40" -o plugins-3.xml
curl -s "https://plugins.qgis.org/plugins/plugins.xml?qgis=4.0" -o plugins-4.xml

Precedenti abbandonati ma un tempo diffusi

Nessun plugin deprecato nell'insieme caricato. Il feed standard non li include: per l'elenco completo vedi plugins.qgis.org/plugins/deprecated/
Dati: plugins.qgis.org/plugins/plugins.xml?qgis=<versione> · Ogni feed è già filtrato lato server dalla versione di download. Carica più feed (es. 3.40 + 4.0) per un quadro completo: vengono fusi per plugin_id tenendo la versione più alta. Nessun dato lascia il browser.
